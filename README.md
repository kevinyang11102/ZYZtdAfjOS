# Python医院信息管理系统 python18

本项目是基于 Django 框架开发的医院信息管理系统，包括医生、患者、病房、科室和药品等管理模块，能够高效地生成药品单据，辅助医院实现信息化管理。

## 技术栈

- 后端：Python（Django 框架）
- 数据库：MySQL
- 前端：HTML + CSS + JavaScript

## 功能模块

- **医生管理**：实现医生信息的添加、修改、查询和删除等功能。
- **患者管理**：对患者信息进行全面的记录和管理。
- **病房管理**：管理病房信息，包括分配、状态更新等。
- **科室管理**：管理医院各个科室的信息和资源。
- **药品管理**：对药品信息进行管理，确保数据的准确性和及时更新。
- **生成药品单据**：根据需求自动生成药品采购单、库存清单等。

## 系统角色

- 系统管理员：负责系统整体配置和用户权限管理。
- 医生：进行患者诊断，查看和更新患者信息。
- 药剂师：管理药品库存，生成和打印药品单据。

## 运行环境

- 操作系统：Linux 或 macOS（推荐），Windows亦可。
- Python 版本：3.6 或以上。
- MySQL 版本：5.7 或以上。

## 部署步骤

1. 准备 Python 环境及相应版本。
2. 安装项目所需依赖。
3. 配置 MySQL 数据库，并导入初始数据。
4. 配置项目设置，包括数据库连接等。
5. 运行项目。
